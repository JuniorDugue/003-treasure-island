```

  print("You can ride the rollercoaster!")
```

```

```
  >You can ride the rollercoaster!

```

  print("You can ride the rollercoaster!")
else:
  print("Sorry, you have to grow taller before you can ride.")
```
  >Sorry, you have to grow taller before you can ride.
